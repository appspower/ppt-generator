# 01. Vision - PPT Generator

> 이 문서는 프로젝트의 북극성입니다. 명시적 피벗 결정 없이 수정하지 않습니다.

## Why We Build This
컨설팅 프로젝트에서 PPT 제작은 가장 시간이 많이 소요되는 작업 중 하나다.
리서치, 구조화, 시각화, 디자인을 자동화하여 **초안의 완성도를 극대화**하고,
컨설턴트가 인사이트와 스토리텔링에 집중할 수 있게 한다.

## Press Release (역방향 정의)
"PPT Generator는 컨설팅 전문가를 위한 AI 기반 프레젠테이션 생성 도구입니다.
주제와 데이터를 입력하면, 맥킨지/BCG 수준의 구조와 프레임워크를 적용한
전문가급 PPT 초안을 자동으로 생성합니다. 레퍼런스 사진 분석, 자동 리서치,
프레임워크 선택, 차트/표 생성까지 원스톱으로 처리합니다."

## Core Principles (비타협 원칙)

### 1. 컨설팅 품질 우선
- 모든 슬라이드는 액션 타이틀 (인사이트 문장형)
- 맥킨지/BCG 스타일의 구조와 레이아웃
- 프레임워크 기반 사고 (2x2 매트릭스, MECE, 피라미드 등)

### 2. 데이터 정직성
- LLM이 숫자를 날조하지 않음
- 차트/표의 데이터는 반드시 실제 소스에서 주입
- 출처 명시 가능한 구조

### 3. 템플릿 기반 브랜드 일관성
- .pptx 마스터 템플릿에서 폰트, 색상, 로고 상속
- 클라이언트별 브랜드 템플릿 지원
- 프로그래밍적 그리드 정렬로 시각적 일관성 보장

### 4. 레퍼런스 학습
- 실제 컨설팅 PPT 캡처를 분석하여 스타일/구조 반영
- 레이아웃 패턴을 학습하고 재현

### 5. 점진적 완성
- MVP → 반복 개선의 사이클
- 처음부터 완벽하지 않아도, 유용한 초안을 빠르게 제공

## Success Criteria
- [ ] 주제 입력 → 10장 이상의 구조화된 PPT 초안 생성
- [ ] 컨설팅 프레임워크 3개 이상 자동 적용 (2x2, 워터폴, 프로세스)
- [ ] 레퍼런스 사진 분석 → 레이아웃 반영
- [ ] 액션 타이틀 자동 생성 (라벨형 제목 0%)
- [ ] 표/차트 포함 슬라이드 생성
- [ ] 마스터 템플릿 기반 브랜드 일관성
