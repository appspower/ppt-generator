# 02. PRD (Product Requirements Document)

> `01_VISION.md`의 핵심 원칙을 기반으로 합니다.

## Target User
- 컨설팅 펌 / 사내 전략팀의 컨설턴트
- PPT 기반 보고서를 자주 제작하는 전문가
- AI 도구를 활용해 초안 품질을 높이고 시간을 절약하려는 사용자

## Core Use Cases

### UC-1: 주제 기반 PPT 생성
- 사용자가 주제/목적을 입력
- LLM이 리서치 + 구조화 수행
- 슬라이드 구성안 (아웃라인) 제안
- 승인 후 전체 PPT 생성

### UC-2: 레퍼런스 기반 스타일 반영
- 사용자가 레퍼런스 PPT 캡처 이미지 업로드
- 이미지 분석으로 레이아웃/구조 패턴 추출
- 추출된 패턴을 신규 PPT에 적용

### UC-3: 데이터 기반 차트/표 삽입
- 사용자가 CSV/Excel 데이터 제공
- 적절한 차트 타입 자동 선택
- 데이터 기반 차트/표 슬라이드 생성

### UC-4: 프레임워크 자동 적용
- 주제에 적합한 컨설팅 프레임워크 자동 선택
- 2x2 매트릭스, 워터폴, SWOT, 프로세스 다이어그램 등
- 프레임워크에 맞춰 내용 구조화

## Information Architecture (화면 흐름)
```
[시작] → [주제/목적 입력] → [아웃라인 생성 & 편집]
  → [레퍼런스 이미지 업로드 (선택)]
  → [데이터 파일 업로드 (선택)]
  → [PPT 생성] → [다운로드]
```
