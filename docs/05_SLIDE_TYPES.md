# 05. Slide Types Catalog

> 지원하는 슬라이드 타입의 상세 정의. 새로운 타입 추가 시 이 문서를 확장합니다.

## 슬라이드 타입 목록

### 1. `title` - 표지
- **용도**: 프레젠테이션 첫 슬라이드
- **요소**: 제목, 부제목, 작성자, 날짜, 회사 로고
- **레이아웃**: 중앙 정렬, 로고 우상단 또는 하단

### 2. `executive_summary` - 요약
- **용도**: 핵심 인사이트 3-5개를 요약
- **요소**: 액션 타이틀, 불릿 포인트 (3-5개), 선택적 강조 박스
- **규칙**: 타이틀은 반드시 인사이트 문장형

### 3. `section_divider` - 섹션 구분
- **용도**: 챕터 간 시각적 구분
- **요소**: 섹션 번호, 섹션 제목
- **레이아웃**: 대형 텍스트, 배경색 또는 이미지

### 4. `chart` - 차트 슬라이드
- **용도**: 데이터 시각화
- **하위 타입**: `bar`, `line`, `pie`, `stacked_bar`, `horizontal_bar`
- **요소**: 액션 타이틀, 차트, 각주/출처
- **규칙**: 차트 타입은 데이터 특성에 맞게 자동 선택

### 5. `table` - 표 슬라이드
- **용도**: 구조화된 데이터 비교/나열
- **요소**: 액션 타이틀, 테이블 (헤더 + 데이터), 각주
- **스타일**: 지브라 스트라이핑, 헤더 강조, 얼터네이트 행 색상

### 6. `two_column` - 2단 레이아웃
- **용도**: 텍스트 + 차트, 텍스트 + 이미지 등 병렬 배치
- **요소**: 액션 타이틀, 좌측 콘텐츠, 우측 콘텐츠
- **변형**: text-text, text-chart, text-image, chart-chart

### 7. `matrix_2x2` - 2x2 매트릭스
- **용도**: BCG, Ansoff, Eisenhower 등 프레임워크
- **요소**: 액션 타이틀, X축/Y축 라벨, 4개 사분면 내용, 선택적 버블/점
- **렌더링**: python-pptx shape 기반 (차트 아님)

### 8. `waterfall` - 워터폴 차트
- **용도**: P&L 워크, EBITDA 브릿지, 증감 분석
- **요소**: 액션 타이틀, 워터폴 차트, 각주
- **렌더링**: matplotlib PNG 임베드

### 9. `process` - 프로세스/타임라인
- **용도**: 단계별 흐름, 로드맵, 마일스톤
- **요소**: 액션 타이틀, 단계 박스들 (좌→우 또는 상→하)
- **렌더링**: python-pptx shape + connector

### 10. `full_image` - 풀블리드 이미지
- **용도**: 임팩트 비주얼, 레퍼런스 사진
- **요소**: 전체 배경 이미지, 오버레이 텍스트 (선택)

### 11. `appendix` - 부록
- **용도**: 보충 데이터, 상세 테이블
- **요소**: 제목, 상세 내용 (텍스트/표/차트)
- **스타일**: 본문보다 정보 밀도 높음

## 차트 타입 선택 가이드

| 데이터 특성 | 추천 차트 |
|---|---|
| 시간에 따른 변화 | 선 차트 (line) |
| 카테고리 비교 | 막대 차트 (bar) |
| 부분-전체 관계 | 파이/스택드 바 (pie/stacked_bar) |
| 순위/크기 비교 | 가로 막대 (horizontal_bar) |
| 순차적 증감 | 워터폴 (waterfall) |
| 2변수 관계 | 산점도/버블 (scatter/bubble) |
| 시장 점유율+규모 | 메코 (mekko) - 향후 지원 |

## 타이포그래피 규칙

| 요소 | 폰트 크기 | 스타일 |
|---|---|---|
| 슬라이드 타이틀 | 20-24pt | Bold |
| 본문 텍스트 | 11-14pt | Regular |
| 각주/출처 | 8-9pt | Italic, Gray |
| 차트 레이블 | 9-11pt | Regular |
| 표 헤더 | 11-12pt | Bold |
| 표 본문 | 10-11pt | Regular |
